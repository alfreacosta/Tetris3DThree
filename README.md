# TetrisThree
Proyecto básico de Tetris desarrollado para la materia Informática 3 en la Universidad Católica.
Este proyecto implementa el juego clásico de Tetris 3D usando HTML y JavaScript.
Trabajo Final - Facultad de la Católica.

## Lenguajes y tecnologías
- HTML
- JavaScript
- Three.js (renderizado 3D)

## ¿Qué hace?
Simula el juego clásico de Tetris en 3D, permitiendo mover y rotar piezas en un tablero tridimensional, sumar puntos y avanzar de nivel.

## Controles
- Flechas: Mover pieza
- Espacio: Caída rápida
- Q/W/E: Rotar eje X
- A/S/D: Rotar eje Y
- Z/X/C: Rotar eje Z
- 2,4,6,8: Rotar cámara
- 5: Centrar cámara
- +/-: Acercar/Alejar
- P: Pausar juego
- R: Reiniciar

## Cómo se juega
El objetivo es acomodar las piezas que caen para formar líneas completas en el tablero 3D. Las líneas completas desaparecen y suman puntos. El juego termina cuando no hay espacio para nuevas piezas.

## Metodología utilizada
Desarrollo incremental y pruebas manuales. Se utilizó Three.js para la visualización y se programó la lógica en JavaScript puro.
